Règles de colorisation CSS

/* Teal (défaut) */
.icon-teal { filter: invert(42%) sepia(60%) saturate(500%) hue-rotate(130deg) brightness(95%); }

/* Warm Accent */
.icon-warm { filter: invert(60%) sepia(80%) saturate(400%) hue-rotate(330deg) brightness(100%); }

/* Innovation (Violet) */
.icon-innovation { filter: invert(50%) sepia(80%) saturate(400%) hue-rotate(220deg) brightness(100%); }

/* Blanc (sur fond sombre) */
.icon-white { filter: brightness(0) invert(1); }

Règles d'Utilisation

À faire et à éviter

✓ À FAIRE

  • • 1 couleur dominante par écran (teal 85%)
  • • CTA toujours teal (#0BA08F ou #00E5A8)
  • • Fond clair = marque / commercial
  • • Fond dark = produit / outil
  • • Accent warm (#FF8F5C) max 10-15%
  • • Icônes SVG colorisées via CSS

✗ À ÉVITER

  • • Multiplier les teals sans logique
  • • Mélanger clair et dark sur un écran
  • • Utiliser des emojis officiels
  • • Accent warm > 15% de la palette
  • • Texte sur dégradé sans plaque
  • • Modifier les couleurs des logos
